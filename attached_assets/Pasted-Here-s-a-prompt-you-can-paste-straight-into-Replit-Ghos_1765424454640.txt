Implement a reusable helper runRouteTest(config: RouteTestConfig): Promise<DiagnosticResult> that:

Measures duration using Date.now() before and after fetch or axios.

Calls the given route on the SAME server (e.g., http://localhost:<port> or use relative fetch if appropriate).

Compares the HTTP status to expectedStatus (defaults to 200).

Returns a DiagnosticResult object with:

status: "PASS" if response is ok and expected status matches

status: "FAIL" if error or wrong status

status: "WARN" if something is off but not fatal (e.g., slow response > 1000ms)

Catches errors and populates error field.

/api/admin/diagnostics/routes endpoint (GET)

Returns the list of available route tests (ROUTE_TESTS) so the frontend can display them.

No testing here, just metadata.

/api/admin/diagnostics/run-all endpoint (POST or GET)

Iterates over all ROUTE_TESTS and runs runRouteTest for each.

Runs them in parallel using Promise.all.

Returns an array of DiagnosticResult.
